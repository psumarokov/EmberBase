EmberBase
=========
Basic Ember REST API Client for fast kick off

Requirements
---------
- Node.js v0.12.2+
- npm install -g ember-cli

Running from CLI
---------
    ember server

Features
--------
- Create, Read, Delete operations for cars
- Image upload
